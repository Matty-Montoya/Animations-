CSS Button Animations

With the installment of CSS3, we were presented with animation and transformation properties.

  • Grow
  • Shrink
  • Rotate
  • Fade

Grow Effects

Click on each button to view the code!

  • HTML
  • CSS
              <button class="btn white grow">
              <p>Grow</p>
              </button>
            
    
    .grow {
    background-color: $redtint;
    border: 1px solid $white;
    border-radius: 5px;
    height: 50px;
    margin: 5px 15px 0;
    padding-top: 15px;
    text-align: center;
    transition: .5s;
    width: 150px;
    }
            
    .grow:hover {
      border: 0;
      box-shadow: 1px 1px 20px 1px $black;
      transform: scale(1.15);
      transition: .5s;
    }

  • Keyframes
                  <button class="btn white fade-out-pulse">
                  <p>Fade-Out Pulse</p>
                  </button>
                
        
        .fade-out-pulse {
          background-color: $greentint;
          border: 1px solid $white;
          border-radius: 5px;
          font-size: 5px;
          height: 50px;
          margin: 15px 15px 0;
          padding-top: 10px;

          animation: pulse 1.5s infinite;
          opacity: .7;
          transition: 1s ease;
        }
                
          
  @keyframes pulse {
    0% {
      box-shadow: 0 0 0 0 rgba(12, 176, 0, .6);
    }

    50% {
      box-shadow: 0 0 0 10px rgba(12, 176, 0, 0);
    }
